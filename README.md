# LeadFlow CRM

A CRM (Customer Relationship Management) project developed using MERN Stack.

---

## Features
- User Authentication Login
- Dashboard Analytics
- Lead Management System
- MongoDB Database Integration
- Responsive UI Design
- Secure Backend APIs

---

## Technologies Used

### Frontend
- ReactJS
- HTML5
- CSS3
- JavaScript

### Backend
- Node.js
- Express.js

### Database
- MongoDB
